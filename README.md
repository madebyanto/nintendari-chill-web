# Nintendari Chill

## Descrizione

Sito vetrina per il server Discord italiano "I Nintendari Chill", dedicato al retro gaming e alle console di generazione Wii/PS3/Xbox 360/3DS.

La pagina offre uno stile neon / cyberpunk con elementi di tipo "CRT" e una UI in puro HTML/CSS/JavaScript, pensata per raccontare la community, i contenuti e le modalità di contatto.

## Funzionalità principali

- Landing page single-page con animazioni CSS e effetto glitch
- Hero section con collegamento diretto al server Discord
- Sezione "Chi siamo" con focus sulla community italiana
- Vetrina dei contenuti: chat, consigli, eventi, scambi, memoria e amicizia
- Sezione contatti con email, Discord e riferimenti TikTok
- Esperienza visiva ispirata alle atmosfere retro e videogame

## Struttura del repository

- `index.html` — pagina principale con HTML, CSS e JavaScript inline
- `assets/` — risorse statiche come il logo

## Note tecniche

- Non sono richieste dipendenze esterne per l'uso base.
- I font vengono caricati tramite Google Fonts.
- Il sito è mobile responsive e ottimizzato per un look sintetico e nostalgico.
